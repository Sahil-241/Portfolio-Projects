# E-Commerce Sales & Inventory Analysis (2024 - 2025)

## 📌 Project Overview
This project focuses on a comprehensive data analysis solution for an E-Commerce business using sales data from 2024 to 2025. The project involves building a relational database schema in SQL for complex data extraction, leveraging MS Excel for data auditing, and establishing a dynamic Power BI dashboard to monitor sales revenue, profit margins, and inventory stock performance.

## 🛠️ Tech Stack Used
* **Database Management:** SQL (MySQL / SQL Server) - Advanced Joins, Window Functions, and CTEs.
* **Data Verification:** MS Excel - Data validation, advanced formulas, and pivot auditing.
* **Business Intelligence:** Power BI - DAX programming, Star Schema data modeling, and interactive executive reporting.

## 📊 Key Insights & Metrics Tracked
* **Sales & Revenue Growth:** Tracked Month-on-Month (MoM) and Year-over-Year (YoY) financial performance.
* **Product & Category Performance:** Identified top-selling products, slow-moving inventory, and high-profit categories.
* **Customer Demographics & Patterns:** Analyzed regional sales distribution and purchase frequencies.
