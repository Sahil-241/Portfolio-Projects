# Call Center Performance & Operations Analytics

## 📌 Project Overview
This project delivers an end-to-end data analytics solution analyzing 2.5 years of operational call center data. By leveraging SQL for data extraction, Excel for data validation, and Power BI for interactive dashboarding, this project identifies critical operational bottlenecks, tracks agent performance, and optimizes workforce management.

## 🛠️ Tech Stack Used
* **Database Management:** SQL (MySQL / SQL Server) - Complex joins, subqueries, and window functions.
* **Data Auditing & Testing:** MS Excel - Pivot tables and data cross-verification.
* **Data Visualization:** Power BI - DAX formulas, Star Schema data modeling, and interactive reporting.

## 📊 Key Insights & Metrics Tracked
* **Service Level Agreement (SLA %):** Tracked daily and hourly SLA compliance against target goals.
* **Call Volume Trends:** Identified peak hours and abandonment rates to optimize staffing.
* **Agent Performance Grid:** Evaluated Average Handle Time (AHT) and Customer Satisfaction (CSAT) scores across teams.
