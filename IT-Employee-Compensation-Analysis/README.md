# IT Employee Compensation & Domain Analysis

## 📌 Project Overview
This project analyzes a dataset of IT professionals across multiple domains (AI, Cyber Security, Web Dev, Data Science, Cloud) to understand salary distribution, experience levels, and remote working trends.

## 🛠️ Tech Stack & Key Columns
* **Tools Used:** MS Excel, SQL
* **Key Columns Covered:** `EmployeeID`, `Name`, `City`, `Domain`, `ExperienceYears`, `MonthlySalary`, `Remote`, `Bonus`, `Total_Compensation`
