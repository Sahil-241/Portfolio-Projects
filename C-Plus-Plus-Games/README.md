# Simple Strategy Games (C++)

## 📌 Project Overview
This folder contains simple, logic-based games like Tic-Tac-Toe, built entirely using C++. The main goal of this project is to practice core programming basics like Arrays, Functions, and User Input validation.

## 🛠️ How it Works & Features
* **Language:** C++
* **Key Logic Handled:**
  * **Real-time Board Update:** Implemented a 2D/Matrix array to display and update the game board instantly after every player's turn.
  * **Input Validation:** Added logical checks to ensure players cannot choose an already filled spot or enter invalid numbers.
  * **Win/Loss Logic:** Created conditions that automatically check rows, columns, and diagonals after every turn to decide the winner or declare a draw.
