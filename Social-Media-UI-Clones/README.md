# 🌐 Social Media Frontend UI Clones Lab

## 📌 Project Overview
This repository contains pixel-perfect, highly responsive frontend user interface replicas of top social media platforms, including **Instagram, Facebook, Twitter (X), Gmail, and WhatsApp Web**. 

The primary objective of this project is twofold:
1. **Frontend Engineering Practice:** To master advanced HTML5 semantics, CSS3 layouts (Flexbox, Grid, custom properties), and media queries for flawless responsive design.
2. **Cybersecurity Awareness (Social Engineering Simulation):** To analyze the structural patterns, visual cues, and layout designs commonly targeted by attackers in credential harvesting and phishing simulations.

---

## ⚠️ Strict Educational Disclaimer
This project is created **strictly for educational purposes, UI/UX research, and cybersecurity awareness**. 
* **Static Frontend Only:** All forms use standard HTML attributes to route to official platforms or simulate actions. There is **NO hidden JavaScript or backend logic** attached.
* **No Data Collection:** This project **does not** harvest, log, or store any user credentials or sensitive data.
* **Ethical Use:** This code must NOT be used for any malicious activities or unauthorized deployment.

---

## 📂 Repository Structure & Tech Stack

Every sub-folder is self-contained with its own clean structural layer (`index.html`) and presentation layer (`style.css`).

* 📸 `/Instagram-Clone` - Modern dark-mode login interface with custom linear gradients.
* 👥 `/Facebook-Clone` - Desktop split-screen layout with responsive landing components.
* 🐦 `/Twitter-Clone` - Authentic dark-theme layout using strict border detailing.
* 📧 `/Gmail-Clone` - Clean, minimalist Google-centric form layout with custom button alignments.
* 💬 `/WhatsApp-Clone` - WhatsApp Web replica featuring a split card grid and dual-colored background split.

### 🛠️ Core Technologies Used:
* **HTML5:** Semantic architecture, form validation tags, structural components.
* **CSS3:** Flexbox, CSS Grid alignment, CSS variables, hover transition states, and Media Queries for cross-device compatibility.
* **FontAwesome:** Scalable vector icons for authentic interface mapping.

---

## 🚀 Key Takeaways & Skills Demonstrated
* **UI/UX Replication:** Ability to convert visual designs into functional, semantic code.
* **Security Mindset:** Practical understanding of how social engineering interfaces are built, helping in better threat detection and phishing analysis during SOC operations.
* **Modular Code Architecture:** Maintaining clean separation between the HTML structural layer and CSS styling files.
