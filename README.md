# My Technical Projects Portfolio

Welcome to my portfolio! Here, I showcase my practical projects across two core engineering domains: **Cybersecurity & Systems Auditing** and **Data Analytics**.

## 🛡️ Cybersecurity & Infrastructure Security Projects

### 1. [SSH Brute-Force Detection & Log Analysis Lab](./SSH-Brute-Force-Log-Analysis-Lab)
* **Tools & Tech:** Linux systemd-journald, `journalctl`, SSH Service Auditing, Kali Linux.
* **Core Concept:** Simulated an active brute-force threat layer using unauthorized authentication vectors. Analyzed systemd internal database engines to isolate indicators of compromise (IoC) and failed credential attempts.

### 2. [Endpoint Activity Monitoring & Local Auditing Tool](./Endpoint-Activity-Monitor-Tool)
* **Tools & Tech:** Python, Operating System APIs, Local Logging Systems.
* **Core Concept:** Developed an automated background utility to capture, monitor, and log endpoint keystrokes and session inputs for administrative auditing and inside-threat compliance.

### 3. [Network Vulnerability Assessment & Asset Mapping](./Network-Vulnerability-Scanner)
* **Tools & Tech:** Nmap (Network Mapper), Nmap Scripting Engine (NSE), Port Scanning Utilities.
* **Core Concept:** Executed network reconnaissance, host discovery, aggressive version fingerprinting, and automated vulnerability correlation mapping against target infrastructures.

### 4. [Network Traffic Sniffing & Deep Packet Analysis](./Wireshark-Network-Traffic-Analysis)
* **Tools & Tech:** Wireshark, Packet Dissection, Network Protocol Filtering.
* **Core Concept:** Captured live enterprise packet streams, applying granular dissection filters to audit unencrypted channels, DNS queries, and detect anomalous layer-2/layer-3 behaviors.

### 5. [Hardware-Based HID Keystroke Injection Lab](./Cybersecurity-Hardware)
* **Tools & Tech:** ATtiny85 (DigiSpark), Microcontroller Scripting, BadUSB Simulation.
* **Core Concept:** Programmed automated Human Interface Device (HID) hardware injection payloads to study USB physical attack vectors and mitigate physical security endpoint risks.
